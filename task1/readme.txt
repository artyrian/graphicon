Линейное растяжение: стандартный алгоритм (может применяться много раз, пока не появится хотя бы одна чёрная точка).

Поканальное растяжение: применим только один раз, т.к. каналы растягиваются независимо. Алгоритм стандартный.

Серый мир: алгоритм  стандартный (из слайдов), хорошие результаты где действительно есть проблемы с балансом белого.


Гаусс(обычный): 2d ядро, нормализация происходит сразу после создания коэффициентов (деление на сумму всех коэффициентов).

Гаусс(сепарабельный): 1d ядро. нормализация происходит по такому же принципу как и в 2d.

Резкость: фильтр Unsharp Mask (при размытии внутри фильтра используется обычный фильтр Гаусса). Для разных фотографий нужно подобрать коэффициент alpha. (Возможно sharpSigma, сейчас = 1.0).

Медианный: алгоритм стандартный (работает и с фильтром чётной размерности).
Опциональный: всё зависит от введёных коэффециентов.


Волны1: стандартный алгоритм (лекции).

Волны2: по бокам не остаётся чёрных волн, т. к. заранее в буфер копирую изображение. Можно сделать как в слайдах на лекции, но решил, что так лучше смотрится.

Стекло: стандартный алгоритм (лекции).


Поворот: работает согласно описанию на форуме (рисуется поверх картинки, пустые места закрашиваются в чёрный цвет).
Масштабирование: перед уменьшением используется сглаживание по Гауссу


Выделение региона: использован класс QRubberBand.
